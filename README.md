# 📝 Advanced To-Do App (Python + Tkinter)

## 📌 Project Overview
Advanced To-Do App is a GUI-based task management application built using Python and Tkinter.  
It helps users efficiently manage daily tasks with features like due dates, priorities, theme switching, and an AI-powered task assistant.

---

## 🚀 Features

### ✅ Task Management
- Add new tasks  
- Delete tasks  
- Mark tasks as completed  

### 📅 Due Date Support
- Tasks include deadlines in **DD-MM-YYYY format**

### 🎯 Priority Levels
- High  
- Medium  
- Low  
(Displayed with different colors)

### 🎨 Modern UI
- Soft pastel **Light Mode**
- Elegant **Dark Mode**

### 🌙 Dark Mode Toggle
- Switch themes dynamically  

### 🤖 AI Task Assistant ⭐
- “✨ Suggest My Day” button  
- Automatically:
  - Reads all tasks  
  - Sorts by priority and due date  
  - Suggests an optimized schedule  

**Example Output:**
✨ Suggested Plan for Your Day:

Submit assignment (High - 27-03-2026)
Study DSA (Medium - 28-03-2026)
Gym (Low - 30-03-2026)

### 💾 File Storage
- Tasks are saved in `tasks.txt`  
- Data persists after closing the app  

### 📊 Smart Sorting
- Tasks are automatically sorted by:
  - Priority  
  - Due date
