# 📝 Advanced To-Do App (Python + Tkinter)

## 📌 Project Overview
Advanced To-Do App is a GUI-based task management application built using Python and Tkinter.  
It allows users to add tasks, assign due dates, set priorities, and manage tasks efficiently.

This project demonstrates concepts such as GUI development, file handling, sorting, and Python data structures.

---

## 🚀 Features

- Add new tasks
- Set due dates for tasks
- Assign priority (High / Medium / Low)
- Sort tasks by due date
- Mark tasks as completed
- Delete tasks
- Persistent storage using a text file
- Simple and user-friendly GUI

---

## 🛠 Technologies Used

- Python
- Tkinter (GUI)
- File Handling
- Datetime module
- Git & GitHub
